Module keyhandler

Helper for dealing with keypresses

This module is a lua script (keyhndlr.lua), you need to explicitly load it with require('keyhndlr')

Class keyhandler

keyhandler:keyhandler () Starts the keyhandler if not already running, returns whether or not it was started
keyhandler:getkeys () Returns a table of all the keys that have been pressed since the last time getkeys was called
keyhandler:stop () Stops the keyhandler
keyhandler:anykey () Blocks until any key is pressed
