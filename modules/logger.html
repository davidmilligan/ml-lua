Module logger

Logging helper.

Writes to a log file and prints to the console.

This module is a lua script (logger.lua), you need to explicitly load it with require('logger')

Usage:

    require('logger')

local log = logger("MYLOG.LOG")
log:write("hello world!")
log:close()
