Module event

Event Handlers.

Scripts can repsond to events by setting the functions in the 'event' table. Event handler functions can take one integer parameter, and must return a boolean that specifies whether or not the backend should continue executing event handlers for this particular event. Event handlers will not run if there's already a script or another event handler actively executing at the same time.

Usage:

    
event.keypress = function(key)

    print("You pressed a key: "..key)

    return false

end
