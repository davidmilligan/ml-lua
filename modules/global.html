beep ([times=1])
Beep

Parameters:

  • times integer number of times to beep (default 1)
call (function[, arg])
Calls an eventproc (a function from the camera firmware which can be called by name). See Eventprocs. Dangerous.

Parameters:

  • function string the name of the function to call
  • arg argument to pass to the call (optional)
shoot ([wait=64[, should_af=true]])
Take a picture

Parameters:

  • wait integer how long to wait for camera to be ready to take a picture (default 64)
  • should_af boolean whether or not to use auto focus (default true)
msleep (amount)
Pauses for ms miliseconds and allows other tasks to run.
